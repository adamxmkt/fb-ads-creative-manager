# ⚡ 快速开始指南

## 5 分钟快速启动

### 第一步：启动应用

```bash
cd /home/ubuntu/fb-ads-creative-manager
./run.sh
```

或者：

```bash
streamlit run app.py
```

### 第二步：打开浏览器

访问：`http://localhost:8501`

### 第三步：开始使用

#### 创建第一个创意

1. 点击左侧菜单的 **✏️ 创建创意**
2. 填写信息：
   - **创意名称**：例如 "夏季促销 - 图片版本"
   - **创意类型**：选择 "图片"
   - **广告账户**：您的 Facebook 账户名
   - **广告活动**：例如 "2月春季活动"
3. 上传一张图片
4. 点击 **💾 保存创意**

#### 查看创意库

1. 点击 **🎨 创意库**
2. 您会看到刚才创建的创意
3. 点击创意名称可以查看详情和预览素材

#### 添加效果数据

1. 点击 **📊 效果追踪**
2. 选择一个创意
3. 输入广告数据：
   - 展示次数：1000
   - 点击次数：50
   - 转化次数：5
   - 花费：$50
4. 点击 **💾 保存效果数据**

#### 查看仪表板

1. 点击 **📊 仪表板**
2. 查看关键指标和最近创意

#### 查看分析

1. 点击 **📈 数据分析**
2. 查看各种图表和统计

## 核心功能一览

| 功能 | 说明 |
|------|------|
| 📊 仪表板 | 查看关键指标和最近创意 |
| ✏️ 创建创意 | 创建新的广告创意和上传素材 |
| 🎨 创意库 | 浏览、搜索和管理所有创意 |
| 📈 数据分析 | 查看创意效果分析和趋势 |
| 📊 效果追踪 | 记录和追踪广告效果数据 |

## 支持的文件类型

**图片：** JPG, PNG, GIF  
**视频：** MP4, MOV, WebM

---

## 常用操作

### 搜索创意
在创意库中使用搜索框快速查找

### 按状态筛选
- **draft** - 草稿
- **running** - 运行中
- **paused** - 暂停
- **ended** - 已结束

### 更新创意状态
在创意库中点击创意，使用下拉菜单更新状态

### 查看素材
在创意库中展开创意，可以预览上传的图片和视频
